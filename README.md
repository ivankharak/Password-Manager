# Passwd-Manager
## Step 1: Installation

```console
# clone the repo
$ git clone https://github.com/Ak500k/Passwd-Manager.git
# change the working directory to Passwd-Manager
$ cd Passwd-Manager

# install the requirements
$ python3 -m pip install -r requirements.txt
```
